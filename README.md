# AI Fake Review & Scam Detector

## Description
Detects whether a message is Fake/Scam or Genuine.

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express

## Run

### Backend
cd backend
npm install
node server.js

### Frontend
Open frontend/index.html

## Example
Input: Win money now! Click link  
Output: Fake/Scam ❌
EOF
